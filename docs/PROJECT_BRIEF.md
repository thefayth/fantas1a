
# Project Brief

Fantasia is a local-first AI operations manager for Faith's laptop, phone lane, and broader project ecosystem. It is designed to conduct agent work the way a strong operator conducts a team: observe current traffic, recommend the next move, route work to the right lane, demand receipts, and keep high-impact authority with Faith.

## Core Promise

Fantasia helps make AI work legible, reversible, and owned.

## Core Product Ideas

- Worksite: a view of current projects, queues, local services, and work traffic.
- Recommendations: start, stop, pause, reassign, review, or escalate.
- Operator Inbox: Codex-optional local packet handling.
- Dev Queue: local development lane with GitTools and scoped authority.
- OpenClaw phone lane: private mobile control path that returns reviewable packets.
- AI Desk: supervised review lane for research, QA, docs, and handoffs.
- Receipts: audit, review, rollback, and status artifacts for every meaningful move.

## Audience

Builders, founders, creators, and operators who want agentic work without losing authorship, safety, or infrastructure control.
