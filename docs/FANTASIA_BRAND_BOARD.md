# Fantasia Brand Board

Fantasia sits under XXYYZZ Society LLC as a premium local-first AI operations desktop. The brand should feel like a private command room: calm, intelligent, receipt-backed, and founder-grade.

## Core Direction

- Positioning: local AI operations desktop for builders who need automation with receipts, rollback, and control.
- Promise: AI with hands, memory, receipts, and brakes.
- Tone: warm operator, sharp strategist, private command room, commercially serious.
- Avoid: cartoon magic, generic SaaS gradients, stock robot imagery, vague productivity language, and anything that makes the product feel unserious.

## Mark Direction

Primary mark concept:

```text
F inside a seal-like command glyph
```

The mark should signal:

- private operator cockpit
- proof seal
- controlled magic
- local desktop authority

Do not use a wand, fairy, cartoon sparkle, or childish fantasy icon.

## Content Pillars

1. Local-first AI operations.
2. Receipts and rollback.
3. FVE website visual workflows.
4. Guarded COAI / WordPress readiness.
5. Owner-controlled automation.
6. Investor-grade trust.
