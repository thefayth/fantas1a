# Contributing

Fantasia is not accepting public source-code contributions at this stage.

This repository is a protected public project surface, not the full source release.

## What Is Welcome

- Public-safe documentation feedback.
- Broken-link reports.
- Accessibility observations.
- Investor/partner questions.
- Suggestions that do not require sharing private source, secrets, customer data, or unpublished workflows.

## What Is Not Accepted

- Pull requests that attempt to recreate or expose private engine behavior.
- Requests for credentials, private prompts, agent instructions, production adapters, or source internals.
- Contributions containing secrets, private data, legal/admin records, medical records, family information, or customer information.

## Commercial Access

Commercial pilots, services, partnerships, licensing, and investor diligence should be handled directly with XXYYZZ Society LLC.
